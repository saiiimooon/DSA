Understanding the AVL Tree

An AVL Tree solves the biggest problem of a standard Binary Search Tree: skewing.

The Balance Factor (BF)

Every node calculates its tilt:
BF = Height(Left) - Height(Right)

A node is balanced only if its BF is -1, 0, or 1.

The Rule of One

If any node's Balance Factor hits 2 or -2, the tree is officially imbalanced.

Java Implementation: The Node & Rotation

class Node {
    int key, height;
    Node left, right;
    
    Node(int d) {
        key = d;
        height = 1;
    }
}

// Right Rotation (LL Case Fix)
Node rightRotate(Node y) {
    Node x = y.left;
    Node T2 = x.right;
    x.right = y;
    y.left = T2;

    return x;
}
